# CS2 Skin Radar 🎯

Live scanner de skins do CS2 abaixo do preço de mercado.

**[→ Abrir Radar](https://spxmiguel.github.io/cs2-skin-radar)**

## Features

- Dados ao vivo via **Skinport API** (atualiza a cada 3 min)
- Compara preço Skinport vs preço de referência Steam
- Filtros: desconto mínimo, preço máximo, condição, categoria, StatTrak
- Links diretos para comprar no Skinport e Steam Market
- Deploy automático via GitHub Actions → GitHub Pages

## Fontes de preço

| Fonte | Status |
|-------|--------|
| Skinport | ✅ Ao vivo |
| Steam Community Market | ✅ Referência (via Skinport) |
| CS.MONEY, DMarket, BitSkins, Waxpeer, Buff163, SkinBaron | 🔜 Em breve |

## Tech

Vanilla HTML/CSS/JS — zero dependências, zero build step.
